### **1️⃣ Переполнение целочисленных типов**

Напиши программу, которая:

1. создаёт переменную типа `int`
2. присваивает ей **максимально возможное значение** для `int`
3. увеличивает её на 1
4. выводит результат в консоль

Ответь **письменно** (здесь и дальше "письменно" - комментарием в коде):

* что ты наблюдаешь при запуске программы
* почему этот результат **не гарантирован стандартом C++**
* как называется такое поведение

### **2️⃣ Unsigned vs signed overflow**

Напиши программу, в которой:

1. есть переменная типа `unsigned int`, равная `0`
2. выполняется операция уменьшения (`unsigned_int--`)
3. результат выводится в консоль

Ответь письменно:

* почему значение стало таким
* по какому правилу выполняется арифметика
* является ли это неопределённым поведением

### **3️⃣ Переполнение при промежуточных вычислениях**

Есть код:

```cpp
int a = 1'000'000;
int b = 1'000'000;
long long result = a * b;
```

Задание:

1. запусти этот код и выведи `result`
2. объясни, почему переполнение происходит **несмотря на `long long`**
3. исправь код так, чтобы переполнение **не происходило**

### **5️⃣ Bool и арифметика + сравнение с логикой**

Напиши программу, где:

1. объявлены две переменные `bool a = true`, `bool b = false`
2. выполняются операции:

   * `a + b`
   * `a * b`
   * `a && b`
   * `a || b`
3. все результаты выводятся в консоль

Ответь письменно:

* почему арифметические операции возвращают **не bool**
* чем принципиально отличается `a + b` от `a || b`
* в каких случаях арифметика с `bool` опасна

### **6️⃣ Вещественные типы, точность и IEEE 754**

Напиши программу, которая:

1. объявляет переменные:

```cpp
double a = 0.1;
double b = 0.2;
```

2. вычисляет сумму `a + b`
3. выводит результат с точностью **20 знаков после запятой**
4. проверяет:

```cpp
(a + b == 0.3)
```

и выводит результат сравнения

Ответь письменно:

* почему результат суммы **не равен точно 0.3**
* как это связано с представлением чисел в формате IEEE 754
* что такое **мантисса и экспонента**
* как корректно сравнивать вещественные числа в C++
