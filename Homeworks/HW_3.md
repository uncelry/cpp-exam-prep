### **Задача 1 — The void type**

Напиши функцию `printSum`, которая принимает два целых числа, **ничего не возвращает**, но выводит их сумму в консоль.

Требования:

* функция должна иметь тип возвращаемого значения `void`
* продемонстрируй её вызов из `main`
* попробуй (закомментировав) вернуть значение из `void`-функции и объясни, почему это ошибка

### **Задача 2 — Type conversion**

Дан код:

```cpp
int a = 7;
int b = 2;
double result = a / b;
```

1. Запусти код и посмотри на результат
2. Исправь программу так, чтобы `result` был равен `3.5`
3. Используй **явное преобразование типа** в стиле C++ (`static_cast`)
4. Коротко в комментарии объясни, почему без приведения типов результат неверный

### **Задача 4 — Memory model, variables**

Напиши программу, в которой:

* есть **глобальная переменная** `g`, инициализированная числом `10`
* есть функция `foo`, в которой объявляется **локальная переменная** `x`, равная `5`

Выведи:

1. значение `g` из `main`
2. значение `x` из `foo`

Ответь в комментарии:

* где живёт `g`
* где живёт `x`
* когда они создаются и уничтожаются

### **Задача 5 — Scope rules**

Дан код:

```cpp
int x = 1;

int main() {
    int x = 2;
    {
        int x = 3;
        std::cout << x << std::endl;
    }
    std::cout << x << std::endl;
}
```

1. Напиши, **какой будет вывод программы**
2. Объясни, почему вывод именно такой
3. Добавь вывод глобальной переменной `x`, используя оператор `::`

### **Задача 6 — Static local variables**

Напиши функцию `callCounter`, которая:

* содержит **static локальную переменную**, инициализированную нулём

* выводит текущее значение

### **Задача 7 — const и volatile**

Дан код:

```cpp
const int a = 10;
int b = 20;
```

1. Создай:

   * указатель на `const int`
   * `const` указатель на `int`
2. Покажи (в виде комментариев), какие операции:

   * разрешены
   * запрещены
3. Ответь письменно:

   * можно ли изменить `a`
   * что произойдёт при попытке изменить `a` через `const_cast`
   * для чего в целом нужен `volatile`
