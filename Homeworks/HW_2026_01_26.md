Домашнее задание по теме **std::vector (C++20)**

В каждой задаче обязательно использовать `std::vector`.
Код должен корректно компилироваться и работать.
Аккуратно оформляй ввод и вывод.

---

### **Задача 1. Инициализация и вывод вектора (базовая)**

Создай вектор целых чисел `std::vector<int>`, содержащий значения от 1 до 5.
Выведи все элементы вектора в одну строку через пробел.

---

### **Задача 2. Ввод вектора с консоли**

С клавиатуры вводится число `n`, затем `n` целых чисел.
Сохрани их в вектор и выведи:

* количество элементов,
* все элементы вектора в обратном порядке.

---

### **Задача 3. Простые операции с вектором**

С клавиатуры вводится `n`, затем `n` целых чисел.
Необходимо:

1. Найти минимальный и максимальный элементы вектора.
2. Вывести их значения.

Запрещено использовать обычные циклы для поиска минимума и максимума — используй алгоритмы STL.

---

### **Задача 4. Модификация вектора с помощью алгоритмов STL**

С клавиатуры вводится `n`, затем `n` целых чисел.
Необходимо:

1. Удалить из вектора все отрицательные числа.
2. Оставшиеся элементы отсортировать по возрастанию.
3. Вывести итоговый вектор.

Обязательно использовать:

* `std::remove_if`
* `std::sort`

---

### **Задача 5. Вектор строк**

С клавиатуры вводится число `n`, затем `n` строк (каждая строка может содержать пробелы).
Необходимо:

1. Найти самую длинную строку.
2. Вывести её длину и саму строку.

---

### **Задача 6. Двумерный вектор (матрица)**

С клавиатуры вводятся два числа `n` и `m`, затем `n × m` целых чисел — элементы матрицы.
Матрицу хранить в виде `std::vector<std::vector<int>>`.

Необходимо:

1. Для каждой строки посчитать сумму элементов.
2. Вывести номер строки (нумерация с 0), у которой сумма максимальна.

---

### **Задача 7. Двумерный вектор + алгоритмы STL (повышенная сложность)**

С клавиатуры вводятся `n` и `m`, затем матрица размером `n × m`.

Необходимо:

1. Для каждой строки отсортировать элементы по возрастанию.
2. Удалить из матрицы все строки, сумма элементов которых равна 0.
3. Вывести итоговую матрицу в виде таблицы.

Подсказка:
используй `std::sort`, `std::accumulate`, `std::remove_if`.

---

❗ **Важно:**

* Следи за инвалидацией итераторов при удалении элементов.
* Старайся максимально использовать алгоритмы STL вместо ручных циклов.
* Код должен быть читаемым и аккуратным.
