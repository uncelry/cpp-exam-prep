### 1. Что такое вывод в консоль

Вывод в консоль — это отправка информации (текста, чисел, результатов вычислений) из программы пользователю в окно терминала.  
В C++ это делается с помощью стандартной библиотеки ввода-вывода.

---

### 2. Основной инструмент — `std::cout`

Для вывода используется объект

```
std::cout
```

из заголовочного файла:

```
#include <iostream>
```

Оператор вывода — `<<` (называется оператор «потока влево» или оператор вставки в поток).

Пример простой программы:

```
#include <iostream>  

int main() {     
	std::cout << "Hello, world!";     
	return 0; 
}
```

---

### 3. Вывод текста и значений

Можно выводить строки, числа, переменные:

```
int a = 10; 
double b = 3.5;  
std::cout << "a = " << a << ", b = " << b;
```

Оператор `<<` можно «сцеплять» сколько угодно раз.  
Тип данных значения автоматически понимается системой потоков C++.

---

### 4. Перевод строки

Существует два распространённых способа переноса строки:

1️⃣ Символ `\n`

```
std::cout << "Hello\n"; 
std::cout << "World\n";
```

2️⃣ Манипулятор `std::endl`

```
std::cout << "Hello" << std::endl; 
std::cout << "World" << std::endl;
```

**Разница:**

- `\n` просто перевод строки
- `std::endl` перевод строки **и принудительная очистка буфера вывода**  
    Обычно `\n` быстрее, а `std::endl` используют, когда важно сразу вывести данные (например, в логах).

---

### 5. Форматирование вывода

Для форматирования чисел, ширины поля, точности используют `<iomanip>`.

```
#include <iostream> 
#include <iomanip>  

double x = 3.1415926;  
std::cout << std::fixed << std::setprecision(2); 
std::cout << "Pi = " << x << std::endl;  // Pi = 3.14
```

Пример ширины поля:

```
int a = 42; 
std::cout << std::setw(10) << a;  // вывод с занятием 10 символов
```

---

### 6. Вывод Unicode и русского текста

Если терминал поддерживает UTF-8 (Linux, большинство современных IDE) — обычно достаточно:

```
std::cout << "Привет, мир!\n";
```

В старых Windows-консолях иногда нужно:

- настроить кодировку консоли
- или использовать wide-строки `std::wcout`, но это зависит от окружения и преподавателя обычно это не требует.

---

### 7. Типичные ошибки студентов

❌ Забыли `#include <iostream>`  
❌ Забыли `using namespace std;` и при этом пишут просто cout (лучше всё же писать std-cout)  
❌ Забыли `;` после выражения вывода  
❌ Думают, что `cout` — функция (это объект потока)

---

### 8. Краткое резюме

- Консольный вывод — через `std::cout`
- Оператор вывода — `<<`
- Перевод строки — `\n` или `std::endl`
- Для форматирования — `<iomanip>`
- Работает через систему потоков C++
