## The C++ language: alphabet, keywords, identifiers, constants/literals, operators. Rules for naming identifiers.

### 1. Алфавит и символы языка

Язык C++ использует **латинский алфавит**, цифры и специальные символы. Основные элементы:

- **Буквы**: a–z, A–Z
- **Цифры**: 0–9
- **Специальные символы**: `+`, `-`, `*`, `/`, `%`, =, `;`, `{}`, `()`, `[]`, `<>` и др.
- **Пробелы, табуляции и переносы строки** считаются разделителями и не влияют на выполнение программы (игнорируются компилятором, кроме случаев, когда важны для синтаксиса).

### 2. Ключевые слова (keywords)

Ключевые слова — это **зарезервированные слова языка**, которые имеют специальное значение и **нельзя использовать как имена переменных или функций**.  
Примеры ключевых слов C++:

- `int`, `float`, `double`, `char`, `bool` — типы данных
- `if`, `else`, `switch`, `case` — условные конструкции
- `for`, `while`, `do` — циклы
- `return`, `break`, `continue` — управление потоком
- `class`, `struct`, `public`, `private`, `protected` — ООП
- `const`, `static`, `volatile` — модификаторы

Всего в C++ около 95 ключевых слов (в C++20 их немного больше из-за новых возможностей).

### 3. Идентификаторы

Идентификатор — это **имя переменной, функции, класса или другого объекта**.

**Правила именования идентификаторов:**

1. Может состоять из букв (a–z, A–Z), цифр (0–9) и символа `_`.
2. Не может начинаться с цифры.
3. Не может совпадать с ключевыми словами.
4. Чувствителен к регистру: `Variable` ≠ `variable`.
5. Желательно давать понятные имена, отражающие назначение, например: `sum`, `counter`, `studentName`.

Примеры корректных идентификаторов:

```
int age; 
double total_sum; 
char _grade;
```

Примеры некорректных:

```
int 2number;   // начинается с цифры 
int if;        // ключевое слово 
int my-variable; // содержит дефис
```

### 4. Константы и литералы

- **Литерал** — это фиксированное значение в коде.
- **Константа** — это значение, которое не меняется после инициализации.

Типы литералов:

1. **Целочисленные**: `42`, `0`, `-7`
2. **Вещественные**: `3.14`, `-0.001`
3. **Символьные**: `'a'`, `'Z'`
4. **Строковые**: `"Hello, world!"`
5. **Булевы**: `true`, `false`

Константы можно создавать через `const`:

```
const double PI = 3.14159; 
const int MAX_USERS = 100;
```

### 5. Операторы

Операторы — это **символы или комбинации символов, которые выполняют действия над данными**.

**Основные группы операторов:**

1. **Арифметические**: `+`, `-`, `*`, `/`, `%`
2. **Присваивания**: =, `+=`, `-=`, `*=`, `/=`
3. **Сравнения**: == , `!=`, `<`, `>`, `<=`, `>=`
4. **Логические**: `&&` (И), `||` (ИЛИ), `!` (НЕ)
5. **Инкремент/декремент**: `++`, `--`
6. **Битовые**: `&`, `|`, `^`, `~`, `<<`, `>>`

Пример использования:

```
int a = 5, b = 3; 
int c = a + b;    // 8 
bool check = a > b; // true 
a += 2;           // a = 7
```
